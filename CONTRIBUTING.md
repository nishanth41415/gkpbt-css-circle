# Contributing

Thanks for considering to contribute to this project!

All sorts of contributions are welcomed including reporting bugs, submitting bug fixes and other patches, proposing features or improvements, and so forth.

## Contributing to This Project

- To report a bug, please open an issue.
- Feel free to submit new feature requests and ideas.
- Improve documentation and add examples.
- Fix bugs or enhance the code using pull requests.

## Code of Conduct

Respect and constructive participation is required when interacting in this project.

## Queries

For any question or suggestion, please open an issue.

Thanks for improving this project!
