# Code of Conduct

## Our Pledge
We pledge to foster an inclusive and respectful environment for all.

## Our Standards
Constructive behavior entails respectfulness, civility, and constructive criticism.
Prohibited behavior entails harassment, discrimination, and abuse.

## Enforcement Authority
Community moderators shall enforce the Code of Conduct.

## Scope
The Code of Conduct applies to all community forums and representations.

## Reporting Violations
Violations can be brought to the attention of community moderators for action.

## Enforcement Guidelines
Moderators will determine the appropriate actions which include warnings, suspensions, or expulsions.

## Attribution
Based on the Contributor Covenant Code of Conduct.
